# OverprimeMatchMaker
Twitter: @RinchaaaMako

Overprimeの交流戦のチーム分けを決めるツールです。

Overprimeで必要になるか不明ですが、古のツールを参考に突貫工事しました。

バグとか不具合とかあっても割り切って使っていただきたく。

## 使い方

名前入れてボタンをポチっとするだけ。

画像が生成されるのでコピーしてdiscordとかに貼り付けて使う想定。
